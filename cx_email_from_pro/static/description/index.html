Customize Email From. Pro Version

Main features

  • Configure custom 'From' addresses for models.
    For example purchase@mycompany.com for all messages sent from Purchase Orders,
    invoicing@mycompany.com for Invoices, project@mycompany.com for Tasks etc
  • Toggle on/off overriding 'From' addresses pre-defined in email templates
  • Remember: your mail server must support sending emails using 'from' addresses you have defined!
